# viranda-store-excel-dashboard
Excel dashboard project with cleaned Viranda store data, interactive charts, and fully connected slicers for dynamic sales analysis and insights.

## 📌 About Project
This is an interactive Excel dashboard project created using the Vrinda Store dataset.  
In this project, I performed data cleaning, data analysis, and created dynamic charts using Pivot Tables and Slicers.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Table
- Pivot Chart
- Slicers
- Sorting & Filtering

---

## 📂 Dataset Columns
- Order ID
- Customer ID
- Gender
- Age
- Age Group
- Date
- Month
- Status
- Channel
- Category
- Quantity
- Amount
- Ship City

---

## 📈 Analysis Done
- Order vs Sales Analysis
- Men vs Women Sales
- Order Status Analysis
- Top 5 States by Sales
- Age Group vs Gender Analysis
- Sales by Channels (Amazon, Myntra, etc.)

---

## 📊 Charts Used
- Bar Chart
- Pie Chart
- Column Chart

---

## 🎛 Dashboard Features
- Interactive Dashboard
- Dynamic Charts
- Slicer Connected Charts
- Easy Data Filtering

---

## 🔍 Key Insights
- Women contributed more sales than men.
- Amazon had the highest sales contribution.
- Adult age group placed maximum orders.
- Some orders were cancelled, returned, or refunded.
